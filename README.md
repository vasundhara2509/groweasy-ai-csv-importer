# 🚀 GrowEasy AI CSV Importer

An AI-powered CRM Lead Analysis Dashboard that intelligently analyzes CSV lead data, assigns lead scores, categorizes leads, and generates downloadable AI reports.

---

## ✨ Features

- Upload CSV files
- Preview uploaded data
- AI-powered lead scoring
- Hot / Warm / Cold categorization
- Dashboard analytics
- Interactive Pie Chart
- Search and Filter leads
- Export AI Analysis Report
- Responsive UI

---

## 🛠 Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Axios
- PapaParse

### Backend
- Node.js
- Express.js

### AI
- Groq API (Llama 3.3 70B)
